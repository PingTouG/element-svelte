安装

npm 安装

推荐使用npm安装，它能更好地和打包工具配合使用。

npm i element-svelte -S

CDN

目前可以通过unpkg.com/element-svelte 获取到最新版本 的资源，在页面上引入js和css文件即可开始使用。

<!-- 引入样式 -->
<link
    rel="stylesheet"
    href="https://unpkg.com/element-svelte/lib/theme-chalk/index.css"
/>
<!-- 引入组件库 -->
<script src="https://unpkg.com/element-svelte/lib/index.js"></script>

我们建议使用CDN引入ElementSvelte的用户在链接地址上锁定版本，以免将 来ElementSvelte升级时受到非兼容性更新的影响。锁定版本的方法请查 看unpkg.com。

如果是通过npm安装，并希望配合打包工具使用，请阅读下一节 ：快速上手。
